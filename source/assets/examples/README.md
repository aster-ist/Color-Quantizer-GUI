# Example Image Source

`example.png` in this folder is a PNG conversion of the following source image:

- Title: `Sunflowers in field flower.jpg`
- Source page: <https://commons.wikimedia.org/wiki/File:Sunflowers_in_field_flower.jpg>
- Direct source file kept in this repo: `sunflowers-public-domain-source.jpg`

According to the Wikimedia Commons file page, the source image was created by Bruce Fritz, U.S. Department of Agriculture, and released to the public domain.

This means the image can be reused and modified for documentation, demos, and repository assets.
